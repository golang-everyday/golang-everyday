## Golang开发技巧

- 多个 if 语句尽量折叠成 switch
- 尽量用 `chan struct{}` 来传递信号, `chan bool` 表达的不够清楚
- 总是把 for-select 换成一个函数
- 分组定义 `const` 类型声明和 `var` 逻辑类型声明
- 为整型常量值实现 `Stringer` 接口
- 用 defer 来检查你的错误
